# Barebones copy of Streamlit's example

### Setup:
```
$ poetry install
$ cd st_mui_multiselect/frontend
$ npm install
```

### Run dev:  
You need to run *both* npm dev server for the JS component frontend as well as streamlit
```
$ cd st_mui_multiselect/frontend
$ npm run start
```
```
$ poetry run streamlit run app.py
```
